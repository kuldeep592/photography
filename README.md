# 💻 Development Guide

This project is intentionally built without frameworks so anyone can easily understand and modify it.

## HTML

All sections are contained inside **index.html**.

```
Hero
Portfolio
About
Awards
Services
Products
Testimonials
Contact
Footer
```

To add a new section:

1. Create a new `<section>`
2. Give it a unique id.
3. Add the navigation link.
4. Style it inside `style.css`.

---

## CSS

The stylesheet follows a design system.

```
Variables

↓

Reset

↓

Typography

↓

Components

↓

Sections

↓

Animations

↓

Responsive Media Queries
```

Most colors can be changed from

```css
:root{

}
```

Variables.

For example

```css
--accent
--bg-primary
--text-primary
```

Changing these updates the entire website.

---

## JavaScript

All interactions are inside

```
script.js
```

Modules include

- Preloader
- Navigation
- Active Navigation
- Mobile Menu
- Portfolio Filter
- Product Tabs
- Scroll Reveal
- Lightbox
- Animated Counter
- Testimonial Slider
- Contact Form
- Smooth Scroll
- Hero Parallax
- Cursor Animation

Each module is separated with comments for easier navigation.
